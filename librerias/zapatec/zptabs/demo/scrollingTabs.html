Scrolling Tabs
  • Use option scrollMultiple to define the scrolling of the tabs.
  • Set to false to have the tabs scroll one tab at at time.
  • Set to true to have multiple tabs scrolled.
  • This option is useful if your full set of tabs is not visible in the container.
  • Scrolling occurs when you click on < or > button.
  • scrollMultiple is set to false, so one tab is scrolled at a time.
