The Zapatec Tabs Features
The Zapatec Tabs allows you to section your workflow into Tabs. All Tabs contents are on the client, so the user experiences an immediate (non-server) response between tabs.
Tabs + AJAX gives you tabbed pages. You can cache all the pages on the client or you can use AJAX to fetch page content from the server on demand.
Tabs uses CSS-based tab styles and allows multiple themes on the same page.
Tabs supports keyboard navigation for tab browsing.
Zapatec Tabs + Zapatec Forms used together offer a wizard-style workflow.
